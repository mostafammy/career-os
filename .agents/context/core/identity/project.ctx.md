# CORE-001 — Project Identity

## What CareerOS Is

A Career Operating System — a decision-support platform that helps ambitious people make exceptional career decisions through clarity, structured execution, and continuously compounding knowledge.

## What CareerOS Is NOT

- A generic task manager
- A note-taking application
- A social network
- A recruiting platform or job board
- A document storage service
- A replacement for calendars or email
- A CRM

## Vision Statement

> Enable every ambitious person to make exceptional career decisions through clarity, structured execution, and continuously compounding knowledge.

## What is Career Capital

The measurable collection of durable assets that increase long-term career opportunities: Knowledge, Skills, Credentials, Portfolio, Network, Reputation, Experience.

## Design Philosophy

CareerOS should feel like a calm, intelligent partner. It reduces uncertainty without removing agency. It simplifies complexity without hiding important information. Technology quietly supports the user instead of demanding attention.

## Decision Lens

Every major product decision must answer:
1. Does this help users make better career decisions?
2. Does it strengthen long-term career growth?
3. Does it reduce unnecessary complexity?
4. Does it help knowledge compound over time?

If the answer is "No" to multiple questions, reconsider.
