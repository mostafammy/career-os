# CareerOS User Stories

**Status:** Canonical
**Version:** 1.0

---

## Purpose
This document translates the CareerOS Vision and User Personas into actionable User Stories. These stories follow the core loop defined in the vision: *Capture → Understand → Decide → Prepare → Execute → Reflect → Learn → Improve*.

**Personas are defined in:** `docs/01-product/user-personas.md`

---

## 1. Capture & Organization

- **As a** Strategic Starter, **I want to** quickly capture a job posting from any web browser **so that** I don't interrupt my browsing flow or lose track of the link.
- **As a** Serial Achiever, **I want to** forward an email about a new fellowship into CareerOS **so that** it is automatically added to my opportunity pipeline.
- **As an** Intentional Pivot, **I want to** automatically extract key dates (deadlines, interview times) from a captured listing **so that** my calendar and reminders are populated without manual data entry.

## 2. Understand & Decide

- **As an** Intentional Pivot, **I want to** score a new opportunity against my customized 5-year career goals **so that** I can confidently decide whether it is worth my time to apply.
- **As a** Strategic Starter, **I want to** view all my captured opportunities in a single pipeline view **so that** I know exactly which applications need my immediate attention today.
- **As an** Intentional Pivot, **I want to** log my reasoning for abandoning an opportunity in a Decision Journal **so that** I can review my decision-making patterns later.

## 3. Prepare & Execute (Knowledge Reuse)

- **As a** Serial Achiever, **I want to** search my entire database of past essays, cover letters, and bios **so that** I can reuse strong, proven content instead of writing from scratch.
- **As a** Strategic Starter, **I want to** link specific "Career Capital" assets (like a portfolio project or a specific resume version) to an opportunity **so that** I have all necessary context gathered when I sit down to apply.
- **As an** Intentional Pivot, **I want to** generate a checklist of required documents for a specific role type **so that** I ensure my application is complete before submission.

## 4. Reflect & Learn

- **As a** Strategic Starter, **I want to** receive a prompt to reflect immediately after an interview **so that** I can document questions asked and areas where I struggled while it is fresh in my memory.
- **As an** Intentional Pivot, **I want to** record feedback from a rejection **so that** I can identify skill gaps and adjust my strategy for future applications.
- **As a** Serial Achiever, **I want to** review a summary of my application outcomes over the past year **so that** I can see which types of grants or fellowships have the highest success rate.

## 5. Compounding Career Capital

- **As an** Intentional Pivot, **I want to** dynamically update my "Master Resume" data when I complete a major project at work **so that** my baseline career capital is always up-to-date.
- **As a** Strategic Starter, **I want to** view a dashboard showing my growing portfolio of skills, documents, and interview experiences **so that** I feel tangible progress even when facing rejections.
